# Hotel-Management-System
A Java-based Hotel Management System supporting room booking, customer details, food orders, billing, and checkout. Built with OOP concepts, serialization for data storage, and multithreading for automated backups. Manages 60 rooms across 4 categories with a clean, modular structure.
